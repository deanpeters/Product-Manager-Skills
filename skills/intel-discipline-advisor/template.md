# Intel Triage Card Template

Copy this to record the outcome of an intel-discipline-advisor session — the triage decision as a
handoff artifact. It's deliberately small: the advisor's whole teaching is that two or three
disciplines matched to the decision beat coverage theater, and the card should fit on one screen
so it actually gets followed.

## Template

```markdown
# Intel Triage: [the question, in your words]

**Date:** [YYYY-MM-DD]
**Decision this feeds:** [what changes based on what we find]
**Situation type:** [suspected move / artifact build-or-refresh / structure puzzle / standing watch]
**Signal already seen:** [what tipped you off — this is 1 discipline already flagged, or "none"]
**Constraints:** [free sources only? geography? recurring time available?]

## The Route
| # | Discipline mix | Cadence | Executing skill | Artifact fed | Why this mix |
|---|---|---|---|---|---|
| 1 | | | | | |
| 2 | | | | | |

**Chosen:** [which recommendation(s), and who runs it]
**First action, this week:** [the single concrete next step]

## Corroboration ladder (for suspected moves)
- Already flagged: [discipline + signal]
- Would corroborate independently: [channels that fail differently from the seen signal]
- Escalation: 1 = watch item · 2 = working hypothesis · 3+ = brief leadership

## The off-ramp check
- [ ] Confirmed this is actually an intel question — not one that win/loss interviews,
      discovery, or your own support tickets answer better. If it is, route there and stop.

## Review date
[When to check whether the route produced anything — a cadence nobody revisits is false confidence]
```

## Quality checks before you call it done

- The route has 2-3 rows, not 8 — a full-table route means the triage didn't happen
- Each row's "why this mix" shows the artifact-mapping logic (the lesson travels with the card)
- The corroborating channels are *independent* of the seen signal — re-detecting the same signal type doesn't stack
- The cadence matches the honest time answer, not the aspirational one
